Generated by [Aakaar](https://github.com/aakaar-sdlc) Autonomous SDLC Platform.

## Quick Start

### Run everything with Docker (recommended)
```bash
docker-compose up --build
```
The app will be available at **http://localhost:3000**

### Manual Setup

## Environment Variables

Copy `.env.example` to `.env` and fill in all values:
```bash
cp .env.example .env
# Edit .env — set DATABASE_URL, API keys, JWT_SECRET, etc.
```

**All keys are REQUIRED before starting the app.** The `.env.example` lists every variable with descriptions.



### Backend (FastAPI — port 8000)
```bash
cd backend
pip install -r ../requirements.txt
uvicorn main:app --reload --port 8000
```


### Frontend (Next.js — port 3000)
```bash
cd frontend
npm install
npm run dev
```


## AI Layer

The AI layer (`ai/`) requires:
- **OPENAI_API_KEY** — for text embeddings (text-embedding-3-small)
- **GEMINI_API_KEY** / **OPENAI_API_KEY** — for answer generation (check `ai/rag.py`)
- **DATABASE_URL** — PostgreSQL with pgvector extension enabled

Enable pgvector in PostgreSQL:
```sql
CREATE EXTENSION IF NOT EXISTS vector;
```

## Requirements

- Python 3.11+ (backend)
- Node.js 18+ (frontend)
- PostgreSQL 15+ with pgvector extension (database)
- Docker + Docker Compose (optional, for one-command startup)

## Notes

- Fill `.env` completely before starting — missing keys cause startup errors.
- Install **all** Python dependencies from `requirements.txt` (includes AI packages).
- The frontend uses Next.js 14 App Router — pages are in `frontend/app/`.
