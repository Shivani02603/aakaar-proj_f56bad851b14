**Project Name:** RAG Document Q&A  
**Domain:** Business Intelligence / Data Analytics  
**Client:** Not specified in BRD  
**Team Size:** Not specified in BRD  
**Timeline:** Not specified in BRD  
**Budget:** Not specified in BRD  
**Primary Users:** Users who need to ask questions about data in Excel files  
**Tech Constraints:** Backend: Python FastAPI. Frontend: Next.js. Database: PostgreSQL with pgvector extension. Deploy: Docker Compose (backend + frontend + postgres services). Use text-embedding-3-small (OpenAI) for embeddings and gemini-2.0-flash (Google Generative AI SDK) for answer generation.

Project Summary: This project involves building a web application that allows users to upload Excel files and ask natural language questions about the data. The system processes the files by splitting the content into text chunks, generating vector embeddings, and storing them for retrieval. When a user asks a question, the system finds relevant data chunks, generates an answer using a language model, and presents the response along with citations showing the source file and data location.
